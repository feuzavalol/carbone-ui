# Carbone UI
This project, currently under development, aims to create a user interface for the carbone-api API 
to facilitate the management of carbon footprints associated with the activities of French engineering schools.

## Deployment 
To clone the repo, simply do: 

```
git clone https://github.com/feuzavalol/carbone-ui.git
```

This project is still in its early stages, but if you want to deploy it, you'll need one of the following: 

### NPM 
If you have npm install in your working environment, simply cd in the project's folder and run:

```
npm run dev
```


### Docker
If you don't have npm, or don't want to download it or use it, you will need docker and docker compose. Then you can run: 

```
docker compose up -d
```
